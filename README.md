# Akshit University Website

A modern, responsive website for Akshit University built with HTML, CSS, and JavaScript.

## Pages

- **index.html** - Home page with university information
- **courses.html** - List of courses offered
- **admission.html** - Admission form
- **login.html** - Student login
- **hostel.html** - Hostel facilities information
- **gallery.html** - Campus gallery
- **staff.html** - Faculty information
- **fees.html** - Course fees structure
- **contact.html** - Contact information
- **dashboard.html** - Student dashboard

## Features

✅ Responsive Design
✅ Modern UI with Gradient Headers
✅ Smooth Animations
✅ Mobile Friendly
✅ Easy Navigation
✅ Professional Layout

## To Deploy

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select main branch as source
4. Your site will be live at: `https://yourusername.github.io/akshit-university`

### Netlify
1. Connect your GitHub repository
2. Deploy automatically on each push

### Other Hosting
- Upload files to your web hosting provider
- Access via your domain name
